# 테마 트렌드 대시보드 프로덕트 정의

## 프로젝트 이름
**테마 트렌드 대시보드**

한 줄 소개: 한국 주식의 신고가 달성 종목과 MTT 템플릿 종목들을 테마별로 분석하여 장 추세를 발견하는 실시간 트렌드 대시보드

---

## 문제 정의

### 해결하려는 문제

1. **정보 분산의 어려움**
   - 장중에 신고가를 달성한 수십 개 종목의 정보를 개별적으로 추적하기 어려움
   - 종목별 뉴스와 재무 정보를 통합적으로 분석할 수 없음

2. **테마 기반 투자 전략의 부재**
   - 신고가 종목들이 어떤 테마(산업, 트렌드)에 속하는지 파악하기 어려움
   - 특정 테마에 속한 종목들의 연관성을 이해하기 어려움

3. **추세 변화 감지의 지연**
   - 강세 테마가 언제 시작되고 언제 약해지는지 정량적으로 측정 불가능
   - 모멘텀 투자 기회를 놓치는 상황이 빈번함

4. **데이터 활용의 제약**
   - HTML 리포트 형식의 수동 자료만 존재
   - 역사적 데이터 추적 및 분석 도구가 부족함

---

## 핵심 기능

### 1. 테마별 일일 RS(상대강도) 집계 및 조회
- 특정 날짜의 테마별 RS 점수를 내림차순으로 조회
- 각 테마의 구성 종목 수, 평균 RS, 누적 변동률 표시
- 테마의 강도를 한눈에 파악 가능

### 2. 신규 급등 테마 탐지
- 기준(5일 이동평균)보다 N 포인트 이상 상승한 테마 자동 탐지
- 신규 장 테마의 등장을 실시간으로 캡처
- 임계값 커스터마이징 지원 (기본값: +10 이상)

### 3. 테마 RS 시계열 추이 분석
- 최대 1년간 특정 테마의 일일 RS 변화 시각화
- 멀티셀렉으로 여러 테마를 동시 비교 가능
- 기간 선택으로 관심 구간 집중 분석

### 4. 지속 강세 종목 탐지
- N일 중 M회 이상 신고가 리스트에 등장한 종목 추출
- "일관되게 강세인 종목"을 객관적으로 필터링
- 투자자 신뢰도가 높은 종목 발굴

### 5. 그룹 액션 탐지
- 신규 등장 테마 + RS 상승 추세를 동시에 만족하는 경우 식별
- "세력이 움직이는 테마"의 신호 포착
- 강세 확산 가능성이 높은 테마 추천

### 6. HTML 리포트 자동 파싱 및 적재
- 52주 신고가 및 MTT 템플릿 HTML 리포트를 자동으로 감지 및 파싱
- BeautifulSoup로 두 가지 리포트 형식 모두 처리
- SQLite 데이터베이스에 정규화하여 저장 (데이터 소스 구분)
- CLI 스크립트로 배치 처리 가능

### 7. 다중 데이터 소스 지원
- **52주 신고가 데이터**: 신고가 달성 종목 기반 테마 분석
- **MTT(Mark Minervini Template) 데이터**: 기술적 패턴 기반 종목 분석
- 프론트엔드에서 데이터 소스 전환 가능한 토글 버튼
- 각 소스별 독립적 시계열 추이 분석

## 타겟 사용자

### 주요 사용자 그룹

1. **단기 트레이더 (Day Traders)**
   - 일 또는 주 단위로 빠른 매매를 반복하는 투자자
   - 신고가 기반 모멘텀 전략 활용
   - 실시간 데이터와 빠른 의사결정 도구 필요

2. **테마 투자자 (Theme Investors)**
   - 특정 산업 또는 트렌드에 펀다멘탈을 기반으로 투자하는 투자자
   - 테마의 강도 변화를 추적하여 진입/청산 신호 포착
   - 역사적 추이 분석 및 비교 분석 도구 필요

3. **자산 운용사 및 펀드 매니저**
   - 대량의 데이터를 기반으로 포트폴리오 구성
   - 테마별 강도 분석으로 섹터 배분 결정
   - 자동화된 데이터 파이프라인 필요

### 사용자 특성

- 한국 주식 시장에 대한 중상 수준 이상의 이해도
- 차트 해석 및 기술 분석 경험이 있는 사용자
- 빠른 데이터 접근과 시각화를 선호
- 모바일 및 웹 양쪽 접근 필요

## 주요 사용 시나리오

### 시나리오 1: 아침 장 준비 (오전 8:30~9:00)
**사용자**: 단기 트레이더

1. 대시보드 홈페이지 접속
2. 전일(어제) 날짜의 "테마별 일일 조회" 보기
3. RS 상위 3개 테마 확인
4. 각 테마의 구성 종목 확인
5. 장 개장 전 매매 전략 수립

### 시나리오 2: 신규 테마 감지 및 진입 판단 (장중)
**사용자**: 테마 투자자

1. "신규 급등 테마" 메뉴에서 오늘 탐지된 테마 확인
2. 관심 테마의 "RS 시계열" 그래프 열기
3. 과거 30일 또는 90일 추이 분석
4. 구성 종목 리스트 확인하여 기업 실적/뉴스 조사
5. 진입 여부 결정

### 시나리오 3: 강세 지속 종목 발굴 (주간 분석)
**사용자**: 자산 운용사

1. "지속 강세 종목" 메뉴에서 "최근 20일 중 5회 이상" 조건 설정
2. 거르러진 종목 리스트 다운로드 또는 추출
3. 해당 종목들이 속한 테마 확인
4. 각 테마의 "RS 시계열"로 장기 추이 분석
5. 포트폴리오 운영 전략 수정

### 시나리오 4: MTT 스크리닝 종목의 테마 집중도 분석 (기술적 분석)
**사용자**: 기술적 분석 투자자

1. 대시보드 헤더의 데이터 소스 토글에서 "MTT 종목" 선택
2. 오늘 날짜로 MTT 템플릿을 만족하는 종목들의 테마별 분포 조회
3. 특정 테마의 "RS 시계열" 그래프로 MTT 종목 집중도 추이 분석
4. 기술적으로 강세인 테마의 모멘텀 강도 파악
5. 기술적 진입점 설정 및 매매 실행

## 차별성

### 경쟁 우위

1. **실시간 테마 강도 분석**
   - 단순 종목 나열이 아닌 정량적 RS 점수 제공
   - 테마 간 강도 비교 가능

2. **자동화된 데이터 파이프라인**
   - 수동 HTML 파싱 제거
   - 일관된 데이터 품질 보장

3. **멀티 스케일 분석**
   - 일일 강도부터 장기 추이까지 다양한 시간축 제공
   - 신규 테마 탐지 + 지속성 분석 + 그룹 액션 종합

4. **사용자 친화적 UI**
   - 직관적인 차트와 테이블
   - 모바일 최적화
   - 빠른 데이터 로딩
