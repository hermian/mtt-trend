## 권장 사항

### 개발 환경 설정

1. **Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

2. **Backend**
   ```bash
   cd backend
   source .venv/bin/activate
   pip install -r requirements.txt
   uvicorn app.main:app --reload
   ```

### 원격 저장소 연결 (선택)

```bash
# GitHub 원격 저장소 추가
git remote add origin https://github.com/username/mtt-trend.git

# 첫 푸시
```
