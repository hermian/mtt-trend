# Report: Above MA Realtime Chart Integration (Double-Pane Sync Mode) (Issue #3)

> 실시간 Above MA(10/20/50) 이동평균선 상회 종목 비율 데이터 연동, 15분 그리드 선형 보간 알고리즘 구축 및 이중 패널 동기화 차트 구현 완료 보고서

## Executive Summary
본 프로젝트는 실시간 데이터베이스(`realtime_above_ma.db`)의 지표를 활용하여 KOSPI, KOSPI 200, KOSDAQ, KOSDAQ 150 시장의 전반적인 열기(Breadth)를 판독할 수 있는 **Above MA 인터랙티브 차트 화면**을 성공적으로 구현 및 배포 완료하였습니다. 가정용 서버 정전으로 발생할 수 있는 5시간가량의 낮 시간대 데이터 유실을 처리하기 위해 정밀한 백엔드 선형 보간 엔진을 함께 탑재하여, 프론트엔드에서 깨짐 없는 유려한 타임라인 분석 환경을 제공합니다.

## 2. 핵심 구현 성과

### 2.1 15분 그리드 선형 보간 백엔드 모듈 개발 (`above_ma_utils.py`)
- **자동 감지**: 동일 날짜 내의 데이터 포인트 간 차이가 20분을 넘을 경우, 정전 등 장애 구간으로 자동 인식합니다.
- **그리드 정렬**: 누락된 첫 시점부터 마지막 시점까지 정확한 15분 단위 시각(예: 09:15:00, 09:30:00 ...)을 분 단위 연산으로 생성합니다.
- **선형 가중 계산**: 시간 경과율에 가중치를 주어 가격 및 비율 지표를 완벽히 메우고, 소수점 둘째 자리까지 반올림 정렬하여 API 데이터를 정제했습니다.
- **완성도 높은 단위 테스트**: `tests/test_api_above_ma.py`에서 모의 정전 시나리오를 구성하여, 누락되었던 20개의 15분 타임프레임이 칼같이 보간 생성되는지 증명했습니다. (`pytest tests/test_api_above_ma.py` 패스 완료)

### 2.2 이중 동기화 캔버스 컴포넌트 (`AboveMaChart.tsx`)
- **Area & Line Combo**: 종가 추세는 그라데이션 필터가 입혀진 Area 차트로, 비율 추세는 RGB 컬러(10MA: Red, 20MA: Green, 50MA: Blue)의 단일 멀티라인 차트로 세련되게 분리 구성하여 시각적 간섭을 원천 차단했습니다.
- **수평축 연동 바인딩**: Lightweight Charts의 논지컬 타임스케일 범위를 공유함으로써 줌 조작이나 마우스 드래그 스크롤 시 두 개의 독립된 패널이 하나의 유기체처럼 함께 움직입니다.
- **반응형 툴팁 헤더**: 두 캔버스에서 마우스 포인터가 지나가는 시점의 종가 및 10/20/50MA 정확한 수치를 캔버스 상단 텍스트 영역에 모노스페이스 서체로 밀도 있게 표시합니다.

### 2.3 다이내믹 라우팅 및 칩 필터링 (`page.tsx` & 사이드바)
- **전용 탭 라우팅**: 사이드바와 모바일 사이드바의 "Above MA" 선택 시 `tab=above_ma` 파라미터가 감지되어 관련 페이지 영역만 깔끔하게 활성화됩니다.
- **다이내믹 테마 스위칭**: 상단의 칩형 버튼 그룹을 사용해 지수 종류를 바꿀 때마다 `useAboveMaData`가 API 데이터를 새로 쿼리해 갱신합니다.
- **시스템 상태 판독기**: 하단에 타겟 심볼 정보와 SQLite 데이터베이스 파일 정보, 보간 활성화 문구(Active)를 배치하여 관리 편의성을 더했습니다.

## 3. 의사결정 이력 (Decision Records)

- **[Architecture - 보간 시점]** 프론트엔드가 아닌 백엔드(Python API)에서 보간 연산을 처리하여 브라우저 CPU 연산 부하를 차단하고 데이터 가공 일관성을 확보했습니다.
- **[UI/UX - 패널 분리]** 가격(수만 단위)과 백분율(0-100)이 하나의 캔버스에 들어올 때 스케일 압착이 발생하는 것을 막기 위해 가격 차트(350px)와 비율 차트(250px)를 위아래 2층 캔버스로 구조화했습니다.
- **[Layout - 헤더 숨김]** 모바일 기기에서의 시각 가용 면적을 넓히기 위해 차트 화면 전환 시 글로벌 대시보드 요약 헤더를 자동 축소(숨김) 처리하여 공간 효율을 극대화했습니다.
